# Utilizacion de la ramas

-Para crear ramas:
```bash
git branch nombre_rama
```

-Cambiar de rama:
```bash
git checkout nombre_rama
```

-Verificar rama actual
```bash
git branch
```

---

### Juntar ramas con la main

-Cambiar a main:
```bash
git checkout main
```

-Traer los cambios
```bash
git merge rama_con_cambios
```
